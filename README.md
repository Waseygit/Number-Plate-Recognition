# ANPR (Automatic Number Plate Recognition)

This project performs **automatic number plate detection and OCR** using:

- **YOLO11** (Ultralytics) for license plate detection  
- **EasyOCR** for text extraction  
- **OpenCV** for video processing  
- **PyTorch + CUDA** for GPU acceleration  

---

##  Features

- Detect license plates in images or videos  
- Extract text from plates using OCR  
- GPU-accelerated detection (NVIDIA)  
- Real-time video processing  
- Works with webcam or video files  

---

##  Tools & Technologies Used

- Python 3.x  
- YOLO11 (Ultralytics)  
- EasyOCR  
- OpenCV  
- NumPy  
- PyTorch / TorchVision  
- CUDA (GPU support)
